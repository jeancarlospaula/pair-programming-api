# Pair Programmer API Server

This API is a back-end of the project https://github.com/lucas3K/PairPrograming

## Documentation

### Product Register Upload
- Request Route: https://product-register-project.herokuapp.com/products/register/upload
- Request Method: **POST**
- Request Format: **JSON**
- Request Format Fields: **{ "input_name" : "input_value", }**


### List of Registered Products
- Request Route: https://product-register-project.herokuapp.com/products/list/?key=XXXXXXXXXXXXXXXX
- Request Method: **GET**
- **Key**: Password to access the route and get the list of products

## Dependencies
- dotenv
- express
- mongoose
- multer
